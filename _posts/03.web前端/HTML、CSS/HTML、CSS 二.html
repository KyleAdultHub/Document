  • 常用的css属性（二）：
    • background属性
      • 属性解释 
        • background属性是css中应用比较多，且比较重要的一个属性，它是负责给盒子设置背景图片和背景颜色的
      • background属性的几个设置项：
        • background-image 设置背景图片地址
          • 将一个图片设置为背景，默认为平铺的形式
        • background-position 设置背景图片的位置
          • 可以在水平方向设置“left”、“center”、“right”，在垂直方向设置“top”、“center”、“bottom”
          • 除了设置这些方位词之外，还可以设置具体的数值，也可以设置相对父元素的百分比。
            • 注意：当使用百分比来定位背景图片的位置的时候，百分比的意思是：背景图片的对应百分比位置和标签的对应百分比位置相对应；
        • background-repeat 设置背景图片如何重复平铺
          • repeat-x  为横向平铺  pepeat-y 为纵向平铺  no-repeat为不进行重复
        • background-color 设置背景颜色
          • 在背景图片下的颜色
        • background-attachment: fixed;  将背景图片设置为相对于浏览器固定，不会随着浏览器滑动而移动（不常用）
          • 备注：可以将background设置项合成一句：
            • background: url(bgimage.gif)   left  center   no-repeat   #00FF00
              • 顺序没有固定的要求，只需要left 在center左边；
    • 其他常用的样式属性：
      • list-style 一般用于设置无序列表中的小圆点，一般把它设为"无"
        • 如：list-style:none  （默认值是squre）
      • overflow 设置块元素溢出内容的处理方式
        • visible 默认值。内容不会被修剪，会呈现在元素框之外。
        • hidden 内容会被修剪，并且其余内容是不可见的，此属性还有清除浮动、清除margin-top塌陷的功能。
        • scroll 内容会被修剪，但是浏览器会显示滚动条以便查看其余的内容。
        • auto 如果内容被修剪，则浏览器会显示滚动条以便查看其余的内容。
      •  设置元素圆角,将元素四个角设置4px半径的圆角 
        • border-radius:4px;
          • 备注：可以用百分比  % 来进行表示圆角的半径是边长的百分之多少；
      • /设置元素透明度,将元素透明度设置为0.3，此属性需要加一个兼容IE的写法

块元素和内联元素的特性
  • 块元素
    • 块元素包含的特性
      • 支持全部的样式
      • 如果没有设置宽度，默认的宽度为父级宽度100%
      • 盒子占据一行、即使设置了宽度
      • 代码换行中间会产生间距
      • 块元素包含默认的属性
        • p标签：含有默认外边距
        • ul：含有默认的上下外边距margin，和左右padding，和默认行首小圆点；
        • h1~h6标签：含有默认的外边距和字体大小
        • body标签：含有默认的外边距
          • body还有一个默认的属性，当直接对body进行设置背景颜色的时候，浏览器会默认将body作为根节点，将整个浏览器的背景都设置为背景颜色，但是如果对html标签设置背景颜色的时候，body的背景就会只为body部分区域的背景色；
  • 内联元素
    • 内联元素包含的特性
      • 不支持宽、高、margin上下、padding上下
      • 宽高由内容决定
      • 盒子并在一行
      • 子元素是内联元素，父元素可以用test-align属性设置子元素水平对其方式
      • 内联元素之间会有固定的间距（如果内联标签都写在一行上就不会产生间距）
      • 有些内联元素包含默认属性
    • 包含默认样式的内联元素
      • a标签：含有的下划线以及文字颜色
      • em、i标签：文字默认为斜体
      • b、strong标签：文字默认加粗

CSS盒子模型
  • 盒子模型的解释
    • 元素在页面中显示成一个方块，类似一个盒子，CSS盒子模型就是使用现实中盒子来做比喻，帮助我们设置元素对应的样式。
  • 盒子模型的样式
    • 把元素叫做盒子，设置对应的样式分别为：盒子的宽度(width)、盒子的高度(height)、盒子的边框(border)、盒子内的内容和边框之间的间距(padding)、盒子与盒子之间的间距(margin)。
  • 盒子的真实尺寸
    • 盒子的width和height值固定时，如果盒子增加border和padding，盒子整体的尺寸会变大，盒子的真实尺寸为
      • 盒子宽度 = width + padding左右 + border左右
      • 盒子高度 = height + padding上下 + border上下
盒模型的使用技巧及常见问题
  • margin解决元素的居中和边框合并问题
    • 设置不浮动的元素相对于父级水平居中（只适用于非浮动的块元素）
      • 第一种方法：margin-left : auto;  margin-right: auto;   设置左右的margin为auto；
      • 第二种方法：margin:x auto;       设置上下的边距为x， 左右的边距为auto；
        • 备注：上下的margin并不会让元素相对于父元素居中；
    • margin负值让元素位移解决边框合并
      • margin可以设置为负值，可以让元素移动boder的距离，来使两个元素的边框合并；
  • 垂直外边距合并
    • 当两个不浮动的元素，它们的垂直外边距margin相遇时，它们将形成一个外边距。合并后的外边距的高度等于两个发生合并的外边距的高度中的较大者；
      • 可以将元素设置为浮动或者只设置元素的margin-top来解决
  • margin-top塌陷
    • 在两个不浮动的盒子嵌套时候，外部盒子没有设置border的时候，内部紧靠top的盒子设置的margin-top会加到外边的盒子上，导致内部的盒子margin-top设置失败;
      • 外部盒子设置一个边框border
      • 外部盒子设置 overflow:hidden

  • 纯数字/纯英文换行的问题
    • 当限定了块元素的长宽后，如果英文或者数字中间没有空格，将默认为一个单词。所以会出现如下情况：
    • 1.一个单词如果在本行容不下，将会超出边框的限制；
    • 2.当一行容不下两个单词的时候，会默认将第二个单词放到下一行；
      • 第一种方法：word-wrap: break-word;    解决了第1个问题，单词容不下会默认换行；
        第二种方法：word-break: break-all;      两个问题都解决了，单词可以出现在同一行，且不会超出边框限制；
  • 解决内联元素换行产生间距的方法
    • 1、去掉内联元素之间的换行，即将内联元素都写在一行上
    • 2、将内联元素的父级设置font-size为0，内联元素自身再设置font-size
相对地址绝对地址
  • 相对地址
    • 相对于引用文件本身去定位被引用的文件地址，以上的例子都是相对地址，相对地址的定义技巧：
    • “ ./ ” 表示当前文件所在目录下，比如：“./pic.jpg” 表示当前目录下的pic.jpg的图片，这个使用时可以省略。
    • “ ../ ” 表示当前文件所在目录下的上一级目录，比如：“../images/pic.jpg” 表示当前目录下的上一级目录下的images文件夹中的pic.jpg的图片。
  • 绝对地址
    • 相对于磁盘的位置去定位文件的地址，比如：<img src="C:\course5\03day\images\001.jpg" alt="图片" /> 绝对地址在整体文件迁移时会因为磁盘和顶层目录的改变而找不到文件，相对地址就没有这个问题。
常用的图片格式
  • 1、psd 
    photoshop的专用格式。
    优点：完整保存图像的信息，包括未压缩的图像数据、图层、透明等信息，方便图像的编辑。
    缺点：应用范围窄，图片容量相对比较大。
  • 2、jpg 
    网页制作及日常使用最普遍的图像格式。
    优点：图像压缩效率高，图像容量相对最小。
    缺点：有损压缩，图像会丢失数据而失真，不支持透明背景，不能制作成动画。
  • 3、gif 
    制作网页小动画的常用图像格式。
    优点：无损压缩，图像容量小、可以制作成动画、支持透明背景。
    缺点：图像色彩范围最多只有256色，不能保存色彩丰富的图像，不支持半透明，透明图像边缘有锯齿。
  • 4、png 
    网页制作及日常使用比较普遍的图像格式。
    优点：无损压缩，图像容量小、支持透明背景和半透明色彩、透明图像的边缘光滑。
    缺点：不能制作成动画
  • 5、总结 
    在网页制作中，如何选择合适的图片格式呢？
    1、使用大幅面图片时，如果要使用不透明背景的图片，就使用jpg图片；如果要使用透明或者半透明背景的图片，就使用png图片；
    2、使用小幅面图片或者图标图片时，使用png图片；如果图片是动画，可以使用gif。
